# Jerry's Website

A personal website built using React (Vite) and using TanStack Router.

## Development

To run development server: `pnpm run dev`
To build: `pnpm run build`

## Testing

Unit tests are written using Vitest, React Testing Library, and Jest. To run: `pnpm run test:unit`
E2E tests are written with Playwright. To run: `pnpm run test:playwright`
